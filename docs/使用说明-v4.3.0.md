> 亲爱的旅行者，欢迎来到枫丹歌剧院。

> 请跟随我的指引，逐幕登台。
> 帷幕已经拉开，灯光为你亮起。

## 序幕：欢迎来到歌剧院

### 这是什么？

理想原生是一台**圣遗物词条优化舞台**。它的工作很简单：

1. 你把自己的角色数据搬上舞台（导入或手动填写）
2. 舞台灯光亮起 —— 引擎计算出你的**期望伤害**
3. 幕后算法登场 —— 在同样的总词条数下，重新分配每一份词条，找到伤害最高的方案
4. 你带着优化方案回到提瓦特，让你的角色打出更高的数字

### 这台舞台的规矩

- **所有计算全在你的浏览器里完成**，角色数据、伤害数字不会发送到任何地方
- 期望伤害公式基于原神公开的伤害计算机制，覆盖 15 个乘区
- 支持直伤、增幅反应（蒸发/融化）、剧变反应、激化、月曜反应全部五条伤害路径
- 115 名角色、226 把武器、61 套圣遗物数据已就位

### 三句话上手

1. 在首页点击「开始配置」
2. 输入 UID 一键导入角色展柜，或手动选择角色
3. 依次填写各个板块，最后点「计算期望伤害」看结果

你也可以点击「同词条重分配」让算法帮你找最优方案，或点击「理想模板」看看理论上的最优面板。

## 第一幕：数据登台（Enka 导入）

### 通过 UID 导入角色展柜

最快捷的方式是直接导入你的游戏展柜数据。

1. 在「Enka 导入」板块，输入你的游戏 UID（9 位数字）
2. 点击「导入」按钮
3. 等待 1–3 秒，展柜中所有角色的数据自动拉取
4. 在下拉菜单中选择你要分析的角色

**导入成功后，以下内容会自动填充**：
- 角色名称和等级
- 武器（含精炼等级）
- 5 件圣遗物（含主词条、副词条数值）
- 套装组合自动识别

### 服务器路由说明

导入系统会自动识别 UID 所属的服务器：
- **天空岛（官服）**：UID 非 5 开头 → 从 Enka 数据源获取
- **世界树（B 服）**：UID 以 5 开头 → 从 MiniGenshin 镜像获取

两个数据源返回的角色信息格式一致，你无需做任何额外操作。

### 如果导入失败

常见原因及解决办法：

| 情况 | 解决办法 |
|------|---------|
| 提示「暂未收录」 | 先去 enka.network 网站输入自己的 UID 查看一次展柜，让数据入库，再回来导入 |
| 提示「网络请求失败」 | 检查网络连接，稍后再试 |
| 提示「API 暂时不可用」 | 上游服务器在维护，等几分钟再试 |
| 加载超时 | B 服 UID 有时 Enka 响应缓慢，会自动切换到 MiniGenshin 快速通道 |

### 手动录入圣遗物

如果不使用导入，也可以手动填写每一件圣遗物。详见「第三幕：圣遗物」。

## 第二幕：角色与武器

### 选择角色

在「选择角色」板块，使用搜索框查找你要分析的角色。支持中文名称搜索。

搜索框中的每个选项会显示：
- 角色元素颜色标记（火/水/风/雷/冰/岩/草/月）
- 中文名称
- 英文名称

选择角色后，系统自动加载该角色的基础属性（生命值、攻击力、防御力、突破属性等），并过滤出该角色可装备的武器类型。

### 设置角色等级

在角色选择下方，通过数字输入框设置角色等级（1–100）。等级影响基础属性和防御乘区计算。

### 选择武器

在「武器配置」板块，搜索并选择武器。武器列表已根据角色武器类型自动过滤（单手剑角色只显示单手剑，以此类推）。

每个武器选项显示：
- 星级（图标 + 数字）
- 基础攻击力
- 副属性类型和数值

选择武器后，右侧信息栏展示武器的完整面板数据。

### 设置精炼等级

点击 R1–R5 芯片选择精炼等级。被选中的精炼等级以高亮水蓝色显示。

精炼等级影响武器的被动效果数值。当精炼等级 > 1 时，被动描述中变化的数值会以金色高亮标出，方便你直接对照填写。

### 武器被动效果模拟

点击「被动效果模拟」展开面板，可以手动填入以下乘区加成（共 11 项）：

| 加成类型 | 输入单位 |
|---------|---------|
| 攻击力% | 百分比（如 20 表示 20%） |
| 防御力% | 同上 |
| 生命值% | 同上 |
| 攻击力（固定值） | 直接输入数值 |
| 防御力（固定值） | 同上 |
| 生命值（固定值） | 同上 |
| 增伤区 | 百分比 |
| 元素精通 | 直接输入数值 |
| 暴击率 | 百分比 |
| 暴击伤害 | 百分比 |
| 羽毛附伤（固定伤害） | 直接输入数值 |

如果武器没有特殊被动效果，面板中会显示提示文字，说明可以直接在对应乘区中填入数值。

### 属性转模

某些武器的被动效果会将一种属性按比例转化为另一种属性（例如：赤沙之杖将元素精通的 52% 转化为攻击力）。

点击「属性转模」区域：
1. 选择「转化来源」（如元素精通）
2. 选择「转化目标」（如攻击力）
3. 输入转化比例（如 0.52）

你可以添加多条转化规则。每次伤害计算时，转模规则会自动应用到面板属性中。

## 第三幕：圣遗物

### 圣遗物槽位

共 5 个槽位，每个槽位需要分别配置：

| 槽位 | 图标名称 | 可选主词条 |
|------|---------|-----------|
| 生之花 | Flower | 固定生命值（不可更改） |
| 死之羽 | Feather | 固定攻击力（不可更改） |
| 时之沙 | Sands | 生命%/攻击%/防御%/元素精通/元素充能效率 |
| 空之杯 | Goblet | 生命%/攻击%/防御%/元素精通/各元素伤害加成/物理伤害加成 |
| 理之冠 | Circlet | 生命%/攻击%/防御%/元素精通/暴击率/暴击伤害/治疗加成 |

点击「圣遗物编辑」展开面板后，为每个槽位选择主词条。主词条决定了该圣遗物提供的核心属性加成。

### 副词条

每个圣遗物最多拥有 4 条副词条。每一条需要：
1. 选择副词条类型（与主词条不可重复——已被主词条占用的类型会自动禁用）
2. 填入数值

副词条数值以**游戏内显示的实际数值**为准（例如：暴击率 3.5% 填 3.5，元素精通 42 填 42）。

点击「添加副词条」按钮增加一条，点击红色删除图标移除。

### 套装组合

点击「套装组合」展开面板。三种模式：

**散搭（无套装）**
不触发任何套装效果。适用于角色没有成套圣遗物、或手动填入其他乘区来等效替代的情况。

**4 件套**
选择一套完整的 4 件套装。系统会自动应用其 2 件套和 4 件套效果。

**2+2**
选择两套不同的圣遗物，各触发其 2 件套效果。两套不可重复。

搜索圣遗物套装名称即可查找。选择后会展示具体的 2 件套 / 4 件套效果描述。

### Enka 导入自动检测

如果通过 Enka 导入了圣遗物数据，系统会自动识别你当前装备的套装组合，并在面板中优先展示（标记为绿色「Enka」标签），同时自动填充到套装选择中。

## 第四幕：天赋与命座

### 天赋模拟

点击「天赋模拟」展开面板。你可以自由填入角色天赋/被动技能提供的各项加成，共 11 个乘区：

| 乘区 | 填写内容 | 示例 |
|------|---------|------|
| 大权区 | 独立乘区加成% | 如克洛琳德的「大权」效果 |
| 月兆区 | 月反应增伤% | 挪德卡莱角色的月兆加成 |
| 增伤区 | 元素/普攻/重击伤害加成% | 如达达利亚的普攻+1 级 |
| 精通区 | 元素精通加成值 | 如阿贝多的天赋提供 125 精通 |
| 攻击力% | 百分比攻击加成 |
| 防御力% | 百分比防御加成 |
| 生命值% | 百分比生命加成 |
| 攻击力（固定值） | 固定攻击力加成 |
| 防御力（固定值） | 固定防御力加成 |
| 生命值（固定值） | 固定生命加成 |
| 羽毛附伤 | 固定伤害值 | 如申鹤的冰翎 |

**注意**：百分比类型的乘区，直接填数字即可，不需要加 `%` 符号。比如 40% 就填 40。

点击「查看天赋详情」可以展开角色的天赋描述作为参考（数据来源于游戏内文本）。

### 命座模拟

与天赋模拟类似，点击「命座模拟」展开面板，填入命座提供的加成：

| 乘区 | 说明 |
|------|------|
| 暴击率 | 如胡桃 2 命提供的额外暴击 |
| 暴击伤害 | 如珐露珊 6 命的暴伤加成 |
| 擢升区 | 特定角色的擢升乘区加成 |
| 减抗 | 角色命座提供的减抗效果 |
| 基础攻击力（白值） | 直接增加角色基础攻击力白值（如玛薇卡 2 命 +200） |
| 其他（增伤/攻%/防%/生%/固定值） | 与天赋相同的剩余乘区 |

**基础攻击力（白值）**是一个特殊的字段：它直接增加角色面板上的白色攻击力数字，而不是绿色加成。白值提升后，所有百分比攻击力加成都会基于更高的基数计算，收益显著高于普通的固定攻击力。目前游戏中仅玛薇卡 2 命拥有此效果。

同样可以点击「查看命座详情」查看每个命座的具体效果描述。

## 第五幕：倍率与反应

### 技能倍率

选择你的技能所依赖的属性类型和倍率数值。

支持单个属性或双属性混合倍率：
- **单属性**：只选一种（例如：攻击力 × 200%），填 200
- **双属性混合**：例如元素战技同时吃攻击力 × 150% 和元素精通 × 300%，则增加两条

可选的倍率属性类型：
- 攻击力（ATK）
- 生命值（HP）
- 防御力（DEF）
- 元素精通（EM）

每条倍率以百分比填入（如 200 表示 200%）。

点击「+ 添加倍率类型」增加第二条。点击红色 × 号移除。

### 元素反应选择

根据你选择的角色元素属性，系统会自动列出该角色可以触发的所有元素反应类型。

点击反应类型芯片选择你要激发的反应：
- **无反应（直伤）**：不触发任何反应
- **蒸发 / 融化**：增幅反应，可选择 1.5 倍或 2.0 倍
- **超载 / 感电 / 超导 / 扩散 / 碎冰**：剧变反应
- **激化 / 蔓激化 / 超激化**：草元素相关反应
- **月曜反应 / 月感电**：挪德卡莱的月神反应体系

反应信息栏会展示当前反应的倍率系数、默认敌人抗性（10%）、默认敌人等级（100）。

## 第六幕：队伍增益

点击「队伍 Buff」展开完整的队伍增益配置面板。

### 辅助角色

多选搜索框，可以同时选择多个辅助角色（如班尼特、万叶、芙宁娜、钟离等）。选中的角色以标签形式展示。

选择辅助角色后，系统会根据其基础数据自动应用相关增益（具体增益需在后续自定义 Buff 区手动填入数值）。

### 辅助圣遗物套装

多选搜索框，选择队伍中辅助角色携带的圣遗物套装效果（如翠绿之影 4 件套的扩散减抗、千岩牢固 4 件套的攻击力加成等）。

### 元素共鸣

从下拉菜单中选择队伍触发的元素共鸣效果：
- 无共鸣
- 热诚之火（火 ×2）
- 净化之水（水 ×2）
- 迅捷之风（风 ×2）
- 强能之雷（雷 ×2）
- 粉碎之冰（冰 ×2）
- 坚定之岩（岩 ×2）
- 蔓生之草（草 ×2）
- 及其他组合共鸣

### 月曜反应增伤

开关控件，启用后可以填入月曜反应增伤百分比（0–100%）。

此功能适用于挪德卡莱角色在触发月神反应时的额外增伤加成。

### 菈乌玛祷歌

仅当选择了月曜反应（月绽放）时出现。配置菈乌玛的祷歌附伤：
1. 选择菈乌玛命座等级（0 命 / 2 命 / 3 命）
2. 填入菈乌玛的元素精通数值
3. 系统实时计算：精通 × 命座倍率 = 祷歌附加伤害

这个伤害会在伤害流程的「祷歌区」中展示。

### 自由输入增益

12 个自由填入的加成字段，用于覆盖或补充上述自动计算的内容：

| 字段 | 说明 |
|------|------|
| 攻击力% | 如班尼特 Q 的攻击加成 |
| 攻击力固定值 | 如九条的固定攻击加成 |
| 增伤% | 如万叶扩散增伤、芙宁娜 Q 的增伤 |
| 暴击率% | 如罗莎莉亚被动的暴击率 |
| 暴击伤害% | 珐露珊 6 命的暴伤加成 |
| 元素精通 | 如迪奥娜 6 命的精通加成 |
| 减抗% | 如翠绿之影 4 件套、钟离护盾 |
| 月反应增伤% | 挪德卡莱相关的月反应增伤 |
| 独立乘区% | 如云堇的额外倍率区 |
| 擢升区% | 特定角色的擢升加成 |
| 羽毛附伤 | 如申鹤冰翎、云堇旗印 |
| 月兆加成% | 挪德卡莱角色的月兆区加成 |

## 第七幕：期望伤害与优化

### 查看角色面板

当你完成以上配置后，右侧面板（桌面端）或底部面板（手机端）会实时显示当前角色的面板属性：

**始终可见**：攻击力、暴击率、暴击伤害
**展开可见**：生命值、防御力、元素精通、元素充能效率、增伤区、基础伤害（如有）

### 计算期望伤害

点击「计算期望伤害」按钮，系统会根据你的全部配置，按照完整的期望伤害公式计算并展示最终数字。

核心伤害流程分为四个层级，点击「期望伤害计算流程」可以逐层查看：

**第一层 — 基础伤害区**
- 攻击力 × 攻击倍率 + 生命值 × 生命倍率 + 防御力 × 防御倍率 + 元素精通 × 精通倍率
- 乘以基础伤害乘区
- 乘以大权区加成
- 加上羽毛附伤

**第二层 — 反应融合（上）**
- 增幅反应系数（蒸发/融化 ×1.5 或 ×2.0）
- 剧变反应基础伤害
- 激化附加伤害
- 月反应倍率 × 月兆区

**第三层 — 反应融合（下）**
- 元素精通加成区（增幅/剧变/月反应分别计算）
- 增伤区加成
- 祷歌附加伤害

**第四层 — 外围乘区**
- 暴击区（暴击率 × 暴击伤害）
- 擢升区
- 防御区（角色等级 vs 敌人等级）
- 抗性区（敌人抗性 − 减抗效果）
- 独立乘区

最终伤害 = 以上所有乘区相乘。

### 同词条重分配（爬山优化）

这是本舞台的核心剧目。

**工作原理**：保持你的圣遗物**总词条数不变**，通过爬山算法在后台 Web Worker 中搜索最优的词条分配方案。结果告诉你：在同样的词条数下，把词条分配给不同的属性，伤害能提升多少。

重分配遵循圣遗物的物理约束：每件圣遗物保持 4 种互不重复的词条类型（不能消灭类型、也不能新增类型），只会对伤害有效的词条类型参与重分配——直伤下的元素精通、没有充能转模时的充能效率等无效词条会被自动锁定，不会出现在优化面板中。

**操作步骤**：
1. 先完成角色、武器、圣遗物、天赋、倍率、Buff 的全部配置
2. 点击「同词条重分配」
3. 在结果面板中可以看到：
   - 当前每项属性的词条数（如暴击率 5.2 条、暴击伤害 7.3 条等）
   - **词条锚定**：点击图钉图标可以锁定某条属性的词条数不变。例如你不想动暴击率，就固定它
   - 底部显示「已固定 X 项 / 可优化 Y 条词条」
4. 点击「开始优化」→ 等待计算 → 查看结果

**结果展示**：
- 优化前 vs 优化后期望伤害对比
- 提升百分比（绿色正数 = 提升，红色负数 = 下降）
- 词条分布对比表（每项属性的优化前后词条数 + 变化量）
- 点击「查看详情」展开乘区分析表，看每个乘区的优化前后词条分配

### 理想模板

理想模板回答一个问题：**「如果我可以自由分配 N 条词条（不限于当前已获得的圣遗物），理论上能打出多高的伤害？」**

**操作步骤**：
1. 设置总词条数（默认为 25 条，最高可设 53 条——对应 5 件全满词条圣遗物的理论上限）
2. **词条锚定**：与前文相同，点击图钉固定某属性的词条数。充能效率始终可锚定，不锚定时不参与分配。底部显示剩余可用词条数
3. 点击「生成理想模板」
4. 查看结果

**约束规则**：理想模板不再是完全自由的词条分配——单类词条最多 35 条（对应 5 件圣遗物每件各 1 初始位 + 5 次强化），总词条不超过 53 条。这保证了生成的面板在物理上是可实现的。

**结果展示**：
- 理论最高期望伤害（N 条词条）
- **理想 vs 当前面板对比表**：8 项属性（生命/攻击/防御/暴击率/暴击伤害/精通/充能/增伤）的理想值和当前值对比，差值用绿色/红色标注
- 理想词条分配清单（每条属性分配了多少条词条）
- 点击「查看详情」展开完整伤害流程

在存档管理界面底部：
- **导出 JSON**：将当前所有配置导出为一个文件，下载到你的电脑
- **导入 JSON**：选择一个之前导出的配置文件，恢复到当前舞台

适用于换设备、备份、或分享给朋友（直接传文件）。

**注意**：分享链接不含圣遗物的具体数值，仅包含类型和词条分配方案。

## 终幕：操作提示合集

### 页面导航

**分步向导**：进入配置页后，你处于一个 7 步的向导流程中。每个板块占据一屏。

- **桌面端**：左侧显示步骤进度条（共 7 步），点击任一步骤直接跳转
- **手机端**：步骤进度条在顶部以紧凑形式显示
- 底部按钮「上一步 / 下一步」在板块间移动
- 鼠标滚轮可以逐板块滚动（一张一卡，自动停靠），也可以手指滑动

**结果板块**：当你点击「计算期望伤害」「同词条重分配」或「理想模板」时，会在第 7 步之后动态插入结果板块。再次计算会替换之前的结果。

### 面板属性自动更新

右侧（桌面端）或底部（手机端）的角色面板会实时反映你的所有配置变化。当你更改角色等级、武器、圣遗物、Buff 等任何参数时，面板数字自动更新。

### 百分比填写的规则

所有百分比字段在内部以小数存储，但你在界面上看到的和填入的都是**百分号前的数字**。例如：
- 暴击率 46.6% → 填 46.6
- 蒸发倍率 1.5 → 填 150（如果你在倍率区填）

面板显示的属性值则是换算后的实际百分比数字。

### 自动填充机制

当你通过 Enka 导入角色时，以下内容会自动同步：
- 角色选择 → 自动选中对应角色
- 角色等级 → 自动填入
- 武器选择 → 自动选中对应武器 + 精炼等级
- 圣遗物套装 → 自动识别并填入套装组合

如果你后续手动更改了角色，自动填充会停止（避免覆盖你的手动选择）。

### 贴纸弹射

选中角色后，偶尔会看到角色的专属贴纸从屏幕中弹出、飘落。这是一个纯装饰性的彩蛋效果，没有实际功能影响。

### 有效词条自动过滤

同词条重分配和理想模板会智能判断每条词条是否对当前配置有效：
- **充能效率**：仅当角色拥有充能转伤害的机制（雷电将军、绝缘之旗印等）时才参与优化
- **元素精通**：选择直伤时自动排除（除非有精通倍率或精通转模）
- **无效词条**：在重分配界面中不会出现，保持原有数值不变

### 更新公告

首页左下角设有更新公告按钮，点击可查看最新版本的改动内容。每次打开页面时默认不弹出，手动静候查阅。

### 移动端适配

在手机上使用时：
- 面板属性显示在底部而非右侧
- 圣遗物编辑器的 5 个槽位变为单列排列
- 步骤导航变为紧凑的顶部横条
- 所有弹窗面板自动适配小屏幕宽度
- 触摸滑动对应桌面端的滚轮导航
